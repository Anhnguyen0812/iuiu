HƯỚNG DẪN

1. Đặt các file cùng một thư mục:
   index.html
   styles.css
   script.js
   anh1.png
   anh2.png
   anh3.png
   anh4.png
   anh5.png (tùy chọn)
   anh6.png (tùy chọn)

2. Mở index.html để xem giao diện.

3. Để tính năng ghi âm hoạt động ổn định, chạy bằng HTTPS hoặc local server:
   - VS Code + Live Server
   - hoặc chạy: python -m http.server 5500
   Sau đó mở http://localhost:5500

4. Chỉnh ngày trong CONFIG ở đầu file script.js:
   relationshipStart: ngày bắt đầu yêu
   nextAnniversary: ngày kỷ niệm tiếp theo

5. Đổi tên cặp đôi, lời nhắn và ngày gửi trực tiếp trong index.html.
